For code review tomorrow we will discuss the R templates that I have created for accessing calls to R functions via the command-line. I will show the R functions for loading QIIME data (qiime/qiime/support_files/R/loaddata.r), an example script that uses this template, 'anosim.r' (attached), and unit test code (attached) for the loaddata functions.

To run these scripts you need the R packages "RUnit" and "vegan". Install by opening R and running:
install.packages("RUnit")
install.packages("vegan")
q()

To run the test code, unzip the attached folder "tests", and run from within that directory:

R --slave --args --source_dir $QIIME_HOME/qiime/support_files/R < run_all_tests.r

where "$QIIME_HOME" is your top-level QIIME directory. You need a current SVN version of QIIME. You can run the example script 'anosim.r' like this:

R --slave --args -d unweighted_unifrac_dm.txt -m map.txt -c Treatment -o anosim --source_dir $QIIME_HOME/qiime/support_files/R < anosim.r
